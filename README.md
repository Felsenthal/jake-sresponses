# Censorship Course — Reading Responses

Template repo for weekly reading responses in the Internet Censorship
and Online Speech course.

## For students — setup (one time, ~1 minute)

1. Click **"Use this template"** at the top of this page → **"Create a
   new repository"**.
2. **Owner:** your GitHub account. **Name:** anything (e.g.
   `censorship-responses`). **Visibility:** **Private**.
3. In your new repo → **Settings → Collaborators** → **Add people** →
   `feamster` → **Read** permission → send invitation.
4. Paste your repo's URL into the Canvas submission (link in the syllabus).

## Weekly workflow

Each week, before the class where the reading is discussed:

1. Create a new file `week-NN.md` (e.g. `week-01.md`, `week-02.md`) —
   copy the response format from the code block below.
2. Fill in the **Summary** and **Questions** sections.
3. Commit and push.

The green check on your latest commit means the file structure is valid.
A red X means you're missing a required section — fix it and push again.

## Response template

```markdown
## Reading Response

**Summary**
1. Briefly summarize the main points of the reading (minimum 1-2 sentences).

**Questions**
2. What questions do you have? (minimum 1-2 sentences)
```

## What counts

- Weekly responses are graded for **completion**, not correctness.
- Late responses eat into the grace-period balance described in the
  [course syllabus](https://noise-lab.github.io/censorship-course/syllabus).
- The instructor pulls your repo automatically before each class meeting —
  no separate submission step is needed after your one-time setup above.
